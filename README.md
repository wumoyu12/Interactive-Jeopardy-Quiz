## Overview

This is an interactive Jeopardy-style quiz game designed for students.

The program includes questions from multiple subjects, including:
- Math
- Physics
- Earth Science
- History

Students can play individually or in teams (1–4 teams), answering questions to earn the points.

## Purpose

The main goal of this project is to:
- Test students' understanding of different subjects
- Encourage active learning through game-based interaction
